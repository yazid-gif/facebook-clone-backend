## üéØ Project Overview

This is the backend API for a Facebook clone application that provides core social media functionalities. It handles user management, authentication, posts, comments, likes, friendships, and messaging features.

## ‚ú® Features

- **User Authentication** - Secure user registration and login with JWT tokens
- **User Profiles** - Create and manage user profiles with profile pictures and bio
- **Posts** - Create, read, update, and delete posts with multimedia support
- **Comments & Replies** - Add comments and replies to posts
- **Likes System** - Like/unlike posts and comments
- **Friend Management** - Send, accept, and manage friend requests
- **Messaging** - Direct messaging between users (if implemented)
- **Notifications** - Real-time notifications for user interactions
- **Search** - Search for users and posts
